This is a web application that allows you to create an account and browse through some of the most famous sorting algorithms and see them in action.

Simply pick the amount and speed of the sortable bars and watch them sort themselves.

I created this project using Python (Flask) for the backend and Javascript for the sorting algorithm visualization.
